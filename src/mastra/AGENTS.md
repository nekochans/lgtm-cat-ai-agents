# src/mastra/ ディレクトリ構成ルール

`src/mastra/` 配下のディレクトリ構成と各フォルダの責務を定義します。

## ディレクトリ概要

| ディレクトリ / ファイル | 責務                                                                                |
| ----------------------- | ----------------------------------------------------------------------------------- |
| `src/mastra/agents`     | エージェントの定義と設定（振る舞い、目標、使用ツール）                              |
| `src/mastra/workflows`  | エージェントとツールを組み合わせたマルチステップワークフローの定義                  |
| `src/mastra/tools`      | エージェントが呼び出せる再利用可能なツールの作成                                    |
| `src/mastra/mcp`        | （任意）外部エージェントとツールを共有するためのカスタム MCP サーバー               |
| `src/mastra/scorers`    | （任意）エージェントのパフォーマンスを評価するスコアラーの定義                      |
| `src/mastra/public`     | （任意）ビルド時に `.build/output` にコピーされ、ランタイムで配信可能になるファイル |
| `src/mastra/index.ts`   | Mastra の設定と初期化を行うエントリーポイント                                       |

## Mastra Skills

Mastra のスキルの利用方法については @AGENTS.md を参照してください。
